Cách xóa hết Migrations và chạy lại:
1. Xóa database
2. Xóa Migrations
3. Chạy lệnh python manage.py runserver để tạo lại database
4. Chạy python manage.py makemigrations -> python manage.py migrate
5. Tạo lại user admin
